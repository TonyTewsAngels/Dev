To use the Learn Easy release 1 demo:

1. Download the file 'RELEASE 1 DEMO LESSON.zip' to your local drive
2. Extract all the files and their structure into a folder
3. Run the 'LearnEasy.jar' executable jar file
4. Open the 'Release1Lesson.xml' lesson file inside the LEarn Easy application


To make changes to the lesson:
1. Comlete steps 1 and 2 as above
2. Open the 'Release1Lesson.xml' lesson file in your favourite text editor
3. Adjust the xml as desired
